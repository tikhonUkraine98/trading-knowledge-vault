# Стакан проти стрічки угод (DOM vs Tape)

## Назва українською

Стакан проти стрічки угод.

## Англійський термін

DOM vs Tape.

## Простою мовою

Стакан показує заявки, тобто наміри.

Стрічка угод показує виконані угоди, тобто дії.

## Ключова ідея

```text
Order Book = наміри
Tape = дії
```

## Що бачить трейдер

- у стакані: де стоять заявки;
- у стрічці: що реально виконалося;
- чи підтверджує стрічка те, що видно у стакані.

## Що бачить бот

- order book snapshots;
- executed trades;
- чи виконуються великі заявки;
- чи зникають вони без виконання.

## Типові помилки

- Довіряти тільки стакану.
- Не перевіряти заявки через виконані угоди.
